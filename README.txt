####Program to assign each member groups and exhibition topics####

***Requirements

- Python:
	https://www.python.org/downloads/

- Powershell

***Instructions:

1. Open Powershell in the folder where the python file is located.
2. The files to be used must be in the same address where the program is.
3. The format of the command should: "python <python_file> <topic_file> <members_file>"
